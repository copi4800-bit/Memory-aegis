# Plan: 081 Aegis Gauntlet

## Intent

Create a first serious stress gauntlet that can answer whether Aegis looks like a strong product candidate rather than only a strong core.

## Workstreams

### 1. Scenario Matrix

- Define one bounded scenario per gauntlet area.
- Keep the scenarios deterministic and local-first.

### 2. Runner

- Build a runner that creates isolated temp DBs.
- Record per-scenario results and grouped summary.

### 3. Verification

- Add tests for artifact shaping.
- Run gauntlet plus full suite.
